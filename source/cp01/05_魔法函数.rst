5 魔法函数
==========

5.1 构造、初始化和删除
~~~~~~~~~~~~~~~~~~~~~~

-  ``__new__``
   用来创建类并返回这个类的实例。注意如果不返回实例的话，相当于实例没有被创建出来。注意这是一个境静态方法，且不需要专门用static来声明，也就是说这时一个特例。
-  ``__init__`` 用来初始化
-  ``__del__`` 相当于析构函数

.. code:: python

   class MyClass:

       def __new__(cls):
           print('in new')
           return super().__new__(cls)

       def __init__(self):
           print('in init')

       def __del__(self):
           print('in del')

   c = MyClass()

上面的代码执行后会输出 s

::

   in new
   in init
   in del
